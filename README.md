## ✨ 주요 기능
- **반응형 웹 디자인:** 데스크탑, 태블릿, 모바일 모든 기기 최적화
- **비주얼 슬라이더:** Swiper.js 기반의 주요 소식 및 이미지 슬라이드
- **복사 기능:** 클릭 한 번으로 대표번호 및 FAX 번호, 주소 클립보드 복사 (토스트 알림 지원)
- **지도 연동:** 카카오맵/네이버 지도 기반의 사무실 위치 안내
- **상담 예약:** 카카오톡 및 네이버 톡톡 상담 바로가기 연동

## 🛠 Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **UI Framework:** Bootstrap 5.3
- **Libraries:** - [AOS](https://michalsnik.github.io/aos/) (Scroll Animation)
  - [Swiper](https://swiperjs.com/) (Slider)
  - [GLightbox](https://biati-digital.github.io/glightbox/) (Image Popup)
  - [Bootstrap Icons](https://icons.getbootstrap.com/)
- **Deployment:** Cloudflare Pages
